使用http-server开启一个本地服务器

前言

http-server 是简单的零配置命令行 http 服务器。

在浏览器运行HTML页面：

file协议：从本地文件夹中直接打开
http协议：利用 node.js 中的 http-server ，开启本地服务器

1、安装 nodejs

官网地址：https://nodejs.org

检查版本，确认是否安装成功。

2、安装 http-server

3、开启 http-server 服务

终端进入目标文件夹，然后在终端输入：

浏览器访问 http://127.0.0.1:8080

4、关闭 http-server 服务

按快捷键CTRL-C
终端显示^Chttp-server stopped.即关闭服务成功。

5、备注

html文件最好命名为index，因为http默认会打开index，这样输入以上两个地址就可以直接进入网页。

通过http-server打开的网站如果需要同步更新，就是修改本地的html、css、js代码后，通过刷新网页就可以实时更新修改后的信息的方法。
