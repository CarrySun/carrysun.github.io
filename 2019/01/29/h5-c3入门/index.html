项目 范例 说明
[if IE] 所有的 IE 可识别
[if IE6] 仅 IE6 可识别
! [if !IE] 除 IE 外都可识别
lt [if lt IE 6] 小于运算符。IE6 以下版本可识别
lte [if lte IE 6] 小于或等于运算。IE6 以及 IE6 以下版本可识别
gt [if gt IE 5] 大于运算符。 IE5 以上版本可识别
gte [if gte IE 5] 大于或等于运算。IE5 以及 IE5 以上版本可识别
( ) [if !(IE 7)] 子表达式。
& [if (gt IE 5)&(lt IE 7)] AND 运算符。
竖线 [if (IE 6) 竖线 (IE 7)] OR 运算符。

原理

  • 在 css 中设置 该标签的 display: block
  • 在 js 中通过 DOM 的方式创建这个标签
