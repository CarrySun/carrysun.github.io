background

background:-color -image -repeat -attachment -position;

- - 说明
^-image 背景图像 background-image:url('paper.gif');
^-repeat 重复默认 repeat repeat repeat-x repeat-y no-repeat inherit
^-attachment 固定/滚动默认 scroll scroll：背景图片随页面的其余部分滚动；fixed：背景图像是固定的；inherit
^-position 起始位置默认 0 0 left top x% y% xpos ypos(px、em...) 左上角是 0 0，右下角是 100 100。如果仅指定了一个值，其他值将是 50％。可混合％和 positions；inherit
